Designer's Comments:

This folder contains images of custom cards, tokens, or other paper gaming components exported from Strange Eons, the gaming component design tool. The number of images depends on the type of component and the options selected at export time. Most gaming components consist of a front and back face image. Decks will include one image for every page in the deck.

All of the images are stored in the PNG (*.png) image file format, which can be read by most image processing and viewing applications. When printing images from this archive, be sure to set either the physical size or the print resolution for the image using the listed values to ensure that the images print correctly.
